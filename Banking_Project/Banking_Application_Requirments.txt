Banking Application Requirments:

General Requirments:
- The application must have a user section.
- The application must have an admin section.
- The application should be able to handle invalid inputs
- The application should use TXT files to store data
- There should be error handling when user inpus data into the application

User Requirments:
- A user should have the option to login to their account.
- To login to an account the user should have to enter an email and password
- A user should only be logged in if the email and password match the user stored in the TXT file
- A user should have the option to create an account.
- A user must enter a valid email to sign up
- A user must enter a password greater than 8 characters to sign up
- A user must enter their choosen password twice when signing up
- The two passwords entered by the user in signing up must match
- Once the sign up requirments are meet their detials should be added to the TXT file
- When creating a new account the application should intalise the account with a balance of zero

- The text file must store the information on each user, their email, password and bank balance

- User should be able to edit their email once logged into their account, with a valid email
- User should be able to edit their passwrod once logged into their account.
- When editing their password they must enter them twice and they must match

- User should be able to add to thier bank account balance
- User should be able to withdraw from their account
- User should be able send money from their account to another user using their email
- A user must not be able to withdraw or transfer more money than they have in their account

- User must be able to logout of their account
- When making changes to their account it should be updated within the TXT files.

Admin requirments:
- Admin should be able to login to their account.

- Admin should be able to edit their email once logged into their account, with a valid email
- Admin should be able to edit their passwrod once logged into their account.
- When editing their password they must enter them twice and they must match

- The admin mustbe able to view all accounts in the banking application
- The admin must be able to edit any email for any usr that they choose within the banking application
- The admin must be able to edit any password for any user that they choose within the banking application
- The admin must be able to edit any balance for any user that they choose within the banking application
- Once a chnage has been made by the admin it should be upated in the TXT file.
